Материалы для подготовки
Доступны тут Материалы для подготовки
1. Поднять локально виртуальное окружение
Делаем это локально - нам потребуется среда, в которой мы будем кастомизировать наш сервис, писать код, тестировать модель, тд. Можно использовать другие инструменты (Poetry, Pipenv), а в качестве дефолтного предлагаю использовать venv
Нам понадобится Python 3.8+
git clone https://github.com/aguschin/art-expert-telegram-bot cd art-expert-telegram-bot python -m venv .venv source .venv/bin/activate pip install "mlem[streamlit,flyio]" cd fastapi-deployment/ # перейдем в папку с моделью
Python
2. Зарегаться во flyio
На флай мы будем деплоить и наш REST API сервис, и нашего Телеграм бота. Так что сначала нужно зарегистрироваться и установить flyctl (CLI тулза для работы с флай)
Go to fly.io and set up an account
Install flyctl using this instruction
Login via flyctl auth login
You also need to provide a credit card, but they won't charge you until you exceed free limits. Поскольку российскую карту привязать нельзя, мы выдадим вам карту для использования.
3. Скачать модель, распаковать её и задеплоить
Для туториала мы будем использовать модель, которую я уже сохранил и запушил в репозиторий (после git clone вы получили ее прямо в репе, и следующие шаги туториала использует её). В конце туториала мы вернемся к обучению модели и посмотрим, как нам обучить свою.
Чтобы модель можно было запускать локально, установим нужные пакеты в текущее виртуальное окружение с помощью mlem build venv :
mlem build venv --model price --current_env True
Python
засервим модель локально
mlem serve fastapi --model price --request_serializer pil_numpy
Python
зайдя на localhost:8080 мы увидим
Чтобы теперь задеплоить модель на fly, нужно выполнить это:
mlem deploy run flyio app \ --model price \ --app_name art-expert-$YOUR_TEAM_NAME # измените имя приложения \ --scale_memory 2048 \ --server fastapi \ --server.request_serializer pil_numpy
Python
зайти и проверить, что приложение висит и доступно на https://art-expert-fastapi.fly.dev (в вашем случае адрес будет использовать имя вашего приложения)
Примеры:
https://art-expert-fastapi.fly.dev для --server fastapi
4. Сделать телеграм бота
Запускаем бота с компа
зарегистрировать бота в https://t.me/BotFather, сохранить токен
set token as Shell var: $ export TELEGRAM_TOKEN=... - it will be read in the bot.py
$ cd ../bot/ # перейдем в папку с ботом
install reqs $ pip install -r requirements.txt
update bot.py: change URL in this line to your fastapi deployment. Your telegram bot will redirect requests to your FastAPI deployment to get predictions. The line to change:
client = HTTPClient(host="https://art-expert-fastapi.fly.dev", port=None)
Python
run python bot.py. This makes your telegram bot work 🎉 and is handy for updates/changes in code you’re going to do. To print logs, you can use print() or import logging; logging.error()
Open your bot in telegram and send /start command. You should get some output greeting you. Now sending pictures to bot should make him return prediction.
Деплоим бота в облако
Note that you can’t it both locally and in cloud - bot will issue warnings and it will be harder to develop it and test code updates. So deploy it to cloud once you’ve done with local iterations and create another bot for testing (or simply remove deployment in fly when you need to debug it locally and then re-deploy the bot). (In the end of bootcamp, a bot deployed will be a plus for DEV nomination, so you need to do this eventually).
set telegram token with $ flyctl secrets set TELEGRAM_TOKEN=$tokenvalue
change appname in fly.toml to your app name (the app will be created automatically on deploy)
When you want for your bot to work even when your computer is offline, deploy the bot with $ flyctl deploy.
5. Обучаем модель
Прогнать ноутбук - версия v5 - это туториал. Там обучается простая модель с названием price. В конце ноутбука она сохраняется и можно её скачать.
ArtExpert
Explore and run machine learning code with Kaggle Notebooks | Using data from Art Price Dataset
https://www.kaggle.com/code/aguschin/artexpert
Советы по обучению:
настоящий эксперт может оценить автора, стоимость, и датировку картины по ее виду. Поэтому на вход fastapi приложения мы подаем только фотографию 🥸 с телеграм ботом таких ограничений нет, но он должен отрабатывать и в случае, когда на вход подается только картинка
изначальный тестовый сет небольшой (~700 изображений). Это значит, что обучить действительно точную модель на них у вас вряд ли получится. Обратите внимание на аугментации картинок - это может несколько помочь. Основной буст качества вы можете получить, добавляя дополнительные данные из интернета. Важный момент: тестовые данные я соскрапил с Sotheby’s (взял те, которые не попали в обучающую выборку). Поэтому оттуда брать примеры для обучения нельзя. В поисках стоимости картин обратите внимание на такие сайты как https://www.christies.com, https://findartinfo.com/, https://bidtoart.com.
